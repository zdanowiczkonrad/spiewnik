# Jezusa ukrytego

- **Przeznaczenie:** klasyczne / tradycyjne
- **Tonacja:** C

```
Jezusa ukrytego mam w Sakramencie czcić,        C e F C e G
wszystko oddać dla Niego, Jego miłością żyć;     C e F C G C
On się nam daje cały, z nami zamieszkał tu,      C F G C G C F G C
dla Jego Boskiej chwały życie poświęćmy Mu;      C F G C G C F G C
wiarą ukorzyć trzeba zmysły i rozum swój,        C a e F C e G
bo tu już nie ma chleba, to Bóg, to Jezus mój!   C e F C G C
```
