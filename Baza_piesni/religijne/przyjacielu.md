# Przyjacielu

- **Przeznaczenie:** uwielbienie / dziękczynienie / adoracja
- **Tonacja:** G

```
   1. Przyjacielu,   G C D
   chcę zostać z Tobą, przy Tobie być
   i nie trzeba, byś mówił coś wystarczy, byś był.
   Ref.: Bo nie ma większej miłości niż ta,   G a C D...
   gdy ktoś życie oddaje bym ja mógł żyć.
   2. Chcę być z tobą, gdy jest mi dobrze i kiedy mi źle.
   Przyjacielu, otwieram przed Tobą serce swe.
```
