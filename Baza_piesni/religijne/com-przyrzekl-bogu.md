# Com przyrzekł Bogu

- **Przeznaczenie:** klasyczne / tradycyjne
- **Tonacja:** C

```
1. Com przyrzekł Bogu przy chrzcie raz, C F G C
Dotrzymać pragnę szczerze.   a F G G7 C
Kościoła słuchać w każdy czas   C F G C
I w świętej wytrwać wierze.   a F G G7 C

O Panie Boże, dzięki Ci,   C F G C
Żeś mi Kościoła otwarł drzwi,   C7 F G C
W Nim żyć, umierać pragnę.   a F G G7 C

2. W Kościele tym jest z Ciałem, Krwią / Bóg pod osłoną chleba.
Swym Ciałem karmi duszę mą, / by żyła w Nim dla nieba.

   W to wierzyć zawsze mocno chcę,
   bo tego Kościół uczy mnie.
   W Nim żyć, umierać pragnę.
```
